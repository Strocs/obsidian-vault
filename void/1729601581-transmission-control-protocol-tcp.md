# Transmission Control Protocol (TCP)

TCP is an internet protocol that allow connect computers networks to share information.
Its bidirectional and ensures information was delivered in the same order that was streamed.
This communiation run via an IP network.

The TCP is a connection-based protocol, that means a sender and a receiver need to estrablish a connection first based on some agreed parameters called `three-way handshake procedure`
The server entablish an open connection between sender and receiver.
